#Mern Starter Templet

```
Starter Templet For MERN stack Development.
```

### Start React Project
```
create-react-app client

--run this commant in root directory
```

### Install Depencencies
```
npm i
npm run client-install
```

### Start Server 
```
npm server
```

### Start Client Server 
```
npm run client
```

### Start Both Server and Client App 
```
npm run dev
```
